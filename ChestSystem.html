Chest System

Project information

  • Category: Unity
  • Project URL: Github/ChestSystem

This is a clone of chest system from clash royale, Made with unity. In this game, When you press the explore button the Game will randomly select a chest type and add to the slot. Each chest has its own properties which is customizable. When press the unlock button. It will start unlocking the chest and there will be a timer. you can quicklyt unlock by giving the required gems for it. If something is already unlocking then you cant unlock anything else and you can add more chests if the slots are filled.

Features

  • Chest spwaning.
  • Can create any no of chests.
  • Total no of slots can be expanded.
  • Hitting pipe decreases score.
  • A Modal window that adjusts itself to fit the message.
  • Observer pattern to send and recive callbacks.
  • State amchine to handle states of the chest (locked, unlocked, unlocking)
